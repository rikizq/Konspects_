#Sets #Python #Множества #Элементы #set #remove #discard

## 1. Что такое множества

Множества (sets) используются для хранения уникальных элементов. Повторяющиеся значения автоматически удаляются.

Пример:

```python
numbers = {1, 2, 3, 4}
```

## 2. Создание множеств

Множества создаются с помощью фигурных скобок или функции `set()`.

Пример:

```python
fruits = {"apple", "banana", "orange"}
```

## 3. Добавление элементов

Для добавления новых элементов используется метод `add()`.

Пример:

```python
numbers = {1, 2, 3}
numbers.add(4)
```

## 4. Удаление элементов

Элементы можно удалять с помощью методов `remove()` или `discard()`.

Пример:

```python
numbers = {1, 2, 3}
numbers.remove(2)
```

## 5. Проверка элементов

Оператор `in` позволяет проверить наличие элемента во множестве.

Пример:

```python
numbers = {1, 2, 3}
print(2 in numbers)
```

## 6. Перебор множества

Элементы множества можно перебирать с помощью цикла `for`.

Пример:

```python
numbers = {1, 2, 3}

for number in numbers:
    print(number)
```

## 7. Объединение множеств

Метод `union()` или оператор `|` объединяют элементы двух множеств.

Пример:

```python
a = {1, 2}
b = {2, 3}

print(a | b)
```

## 8. Пересечение множеств

Метод `intersection()` или оператор `&` возвращают общие элементы множеств.

Пример:

```python
a = {1, 2, 3}
b = {2, 3, 4}

print(a & b)
```

## 9. Разность множеств

Метод `difference()` или оператор `-` возвращают элементы первого множества, которых нет во втором.

Пример:

```python
a = {1, 2, 3}
b = {2, 3, 4}

print(a - b)
```

## 10. Практическое использование множеств

Множества используются для удаления дубликатов и выполнения операций над наборами данных.

Пример:

```python
numbers = [1, 1, 2, 2, 3]
unique = set(numbers)

print(unique)
```

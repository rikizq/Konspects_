#Type_Casting

## 1. Преобразование в `int`

Функция `int()` используется для преобразования значения в целое число.

Пример:

```python
age = "18"
age = int(age)
```

## 2. Преобразование в `float`

Функция `float()` используется для преобразования значения в число с плавающей точкой.

Пример:

```python
price = "19.99"
price = float(price)
```

## 3. Преобразование в `str`

Функция `str()` используется для преобразования значения в строку.

Пример:

```python
age = 18
text = str(age)
```

## 4. Преобразование в `bool`

Функция `bool()` используется для преобразования значения в логический тип данных.

Пример:

```python
print(bool(1))
print(bool(0))
```

## 5. Неявное преобразование типов

Python может автоматически преобразовывать некоторые типы данных при выполнении операций.

Пример:

```python
x = 5
y = 2.5

print(x + y)
```

## 6. Явное преобразование типов

Явное преобразование выполняется программистом с помощью специальных функций.

Пример:

```python
num = "25"
print(int(num) + 5)
```

## 7. Преобразование пользовательского ввода

Данные, полученные через `input()`, имеют тип `str`, поэтому часто требуют преобразования.

Пример:

```python
age = int(input("Возраст: "))
```

## 8. Практическое использование преобразования типов

Преобразование типов позволяет выполнять вычисления, сравнения и обработку данных разных типов.

Пример:

```python
a = "10"
b = "20"

print(int(a) + int(b))
```

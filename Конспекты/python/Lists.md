### Введение

Список (List) - изменяемая коллекция данных для хранения нескольких значений в одной переменной. Элементы списка записываются в квадратных скобках и разделяются запятыми. В списке могут храниться данные разных типов: числа, строки, логические значения и другие объекты.

### Доступ к элементам

Каждый элемент списка имеет индекс. Нумерация начинается с `0`, поэтому первый элемент имеет индекс `0`, второй - `1` и так далее. Также можно использовать отрицательные индексы для доступа к элементам с конца списка.

### Изменение элементов

Списки являются изменяемым типом данных, поэтому их содержимое можно менять после создания. Для изменения элемента достаточно обратиться к нему по индексу и присвоить новое значение.

### Добавление и удаление элементов

Для добавления новых элементов используется метод `append()`, который добавляет значение в конец списка. Метод `insert()` вставляет элемент в указанную позицию. Для удаления применяются методы `remove()`, `pop()` и оператор `del`. Метод `remove()` удаляет элемент по значению, `pop()` удаляет элемент по индексу и возвращает его, а `del` удаляет элемент или диапазон элементов.

### Срезы списков

Срезы позволяют получать часть списка. Они записываются в формате `[начало:конец]`, где начальный индекс включается в результат, а конечный нет. При необходимости можно указать шаг выборки.

### Перебор списков

Для обработки элементов списка чаще всего используется цикл `for`. Он позволяет последовательно перебрать каждый элемент коллекции без обращения к индексам.

### Основные методы списков

Списки имеют множество встроенных методов. Наиболее часто используются `append()`, `insert()`, `remove()`, `pop()`, `sort()` и `reverse()`. Они позволяют добавлять, удалять, сортировать и изменять порядок элементов.

### Проверка наличия элементов

Для проверки наличия элемента используется оператор `in`. Он возвращает `True`, если элемент найден в списке, и `False`, если его нет.

### Практическое применение

Списки используются для хранения и обработки наборов данных: пользователей, товаров, оценок, сообщений, результатов вычислений и других объектов. Благодаря изменяемости и большому набору методов они являются одной из основных структур данных в Python.

```python
students = ["Alex", "John", "Kate"]

students.append("Mike")
students.remove("John")

for student in students:
    print(student)

print(len(students))
```

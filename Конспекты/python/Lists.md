## 1. Что такое списки

Списки используются для хранения нескольких значений в одной переменной. Элементы списка располагаются в квадратных скобках и разделяются запятыми.

Пример:

```python
numbers = [1, 2, 3, 4, 5]
```

## 2. Доступ к элементам списка

Каждый элемент списка имеет свой индекс, начиная с 0. Индексы позволяют получать отдельные элементы списка.

Пример:

```python
fruits = ["apple", "banana", "orange"]
print(fruits[0])
```

## 3. Изменение элементов списка

Значения элементов списка можно изменять по индексу.

Пример:

```python
fruits = ["apple", "banana", "orange"]
fruits[1] = "grape"
```

## 4. Добавление элементов

Для добавления новых элементов используется метод `append()`.

Пример:

```python
numbers = [1, 2, 3]
numbers.append(4)
```

## 5. Удаление элементов

Элементы можно удалять с помощью методов `remove()`, `pop()` или оператора `del`.

Пример:

```python
numbers = [1, 2, 3]
numbers.remove(2)
```

## 6. Срезы списков

Срезы позволяют получать часть списка по диапазону индексов.

Пример:

```python
numbers = [1, 2, 3, 4, 5]
print(numbers[1:4])
```

## 7. Перебор списка

Списки часто используются вместе с циклом `for` для обработки каждого элемента.

Пример:

```python
numbers = [1, 2, 3]

for number in numbers:
    print(number)
```

## 8. Основные методы списков

Списки имеют встроенные методы для работы с данными, такие как `append()`, `insert()`, `remove()`, `sort()` и `reverse()`.

Пример:

```python
numbers = [3, 1, 2]
numbers.sort()
```

## 9. Проверка элементов

Оператор `in` позволяет проверить наличие элемента в списке.

Пример:

```python
numbers = [1, 2, 3]
print(2 in numbers)
```

## 10. Практическое использование списков

Списки используются для хранения, обработки и организации наборов данных различного типа.

Пример:

```python
students = ["Alex", "John", "Kate"]
print(len(students))
```

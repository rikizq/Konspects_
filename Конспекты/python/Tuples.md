#Tuples #Python #Кортежи

## 1. Что такое кортежи

Кортежи (tuples) используются для хранения нескольких значений в одной переменной. В отличие от списков, кортежи являются неизменяемыми.

Пример:

```python
numbers = (1, 2, 3, 4, 5)
```

## 2. Создание кортежей

Кортежи создаются с помощью круглых скобок. Они могут содержать данные разных типов.

Пример:

```python
person = ("Alex", 17, True)
```

## 3. Доступ к элементам кортежа

Каждый элемент кортежа имеет индекс, начиная с 0. Индексы используются для получения значений.

Пример:

```python
fruits = ("apple", "banana", "orange")
print(fruits[1])
```

## 4. Срезы кортежей

Срезы позволяют получать часть кортежа по диапазону индексов.

Пример:

```python
numbers = (1, 2, 3, 4, 5)
print(numbers[1:4])
```

## 5. Неизменяемость кортежей

После создания кортежа его элементы нельзя изменить, добавить или удалить.

Пример:

```python
numbers = (1, 2, 3)
# numbers[0] = 10
```

## 6. Перебор кортежей

Кортежи можно перебирать с помощью цикла `for`.

Пример:

```python
numbers = (1, 2, 3)

for number in numbers:
    print(number)
```

## 7. Основные функции для кортежей

Для работы с кортежами часто используются функции `len()`, `max()`, `min()` и `sum()`.

Пример:

```python
numbers = (1, 2, 3)
print(len(numbers))
```

## 8. Проверка элементов

Оператор `in` позволяет проверить наличие элемента в кортеже.

Пример:

```python
numbers = (1, 2, 3)
print(2 in numbers)
```

## 9. Распаковка кортежей

Элементы кортежа можно сразу сохранить в несколько переменных.

Пример:

```python
person = ("Alex", 17)

name, age = person
```

## 10. Практическое использование кортежей

Кортежи используются для хранения данных, которые не должны изменяться во время работы программы.

Пример:

```python
coordinates = (10, 20)
print(coordinates)
```

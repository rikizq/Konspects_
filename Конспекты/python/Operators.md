## 1. Арифметические операторы

Арифметические операторы используются для выполнения математических вычислений.

Пример:

```
a = 10
b = 5
print(a + b)
```

## 2. Оператор присваивания

Оператор `=` используется для сохранения значения в переменной.

Пример:

```
name = "Alex"
```

## 3. Операторы сравнения

Операторы сравнения позволяют сравнивать значения и получать результат `True` или `False`.

Пример:

```
print(10 > 5)
```

## 4. Логические операторы

Логические операторы используются для объединения нескольких условий.

Пример:

```
print(10 > 5 and 7 > 3)
```

## 5. Операторы принадлежности

Операторы `in` и `not in` проверяют наличие элемента в последовательности.

Пример:

```
print("P" in "Python")
```

## 6. Операторы идентичности

Операторы `is` и `is not` проверяют, являются ли объекты одним и тем же объектом.

Пример:

```
a = 10
b = a
print(a is b)
```

## 7. Приоритет операторов

Приоритет определяет порядок выполнения операций в выражении. Для изменения порядка используются скобки.

Пример:

```
print((2 + 3) * 4)
```

## 8. Комбинированные операторы присваивания

Операторы `+=`, `-=`, `*=`, `/=` позволяют выполнять вычисление и присваивание одновременно.

Пример:

```
x = 5
x += 3
print(x)
```

## 1. Переменные

Переменные используются для хранения данных в программе. Им можно присваивать различные значения и изменять их во время выполнения программы.

Пример:

```
name = "Alex"
age = 17
```

## 2. Целочисленный тип данных (`int`)

Тип `int` используется для хранения целых чисел без дробной части.

Пример:

```
count = 10
year = 2025
```

## 3. Вещественный тип данных (`float`)

Тип `float` предназначен для хранения дробных чисел.

Пример:

```
price = 19.99
height = 1.82
```

## 4. Строковый тип данных (`str`)

Тип `str` используется для хранения текстовой информации.

Пример:

```
name = "Python"
message = "Hello"
```

## 5. Логический тип данных (`bool`)

Тип `bool` содержит два значения: `True` и `False`.

Пример:

```
is_student = True
is_admin = False
```

## 6. Определение типа данных

Функция `type()` позволяет узнать тип переменной или значения.

Пример:

```
age = 17
print(type(age))
```

## 7. Преобразование типов данных

Функции преобразования позволяют менять тип данных значения.

Пример:

```
age = "17"
age = int(age)
```

## 8. Константы

Константы используются для хранения значений, которые не должны изменяться. Обычно их записывают заглавными буквами.

Пример:

```
PI = 3.14159
MAX_USERS = 100
```

## 9. Работа с данными

Переменные можно использовать в вычислениях и выражениях.

Пример:

```
a = 5
b = 3
print(a + b)
```

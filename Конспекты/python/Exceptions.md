#Exceptions #Python #Исключения #try #except #else #finally

## 1. Что такое исключения

Исключения возникают во время выполнения программы при появлении ошибки. Они позволяют обнаруживать и обрабатывать ошибки без завершения программы.

Пример:

```python
print(10 / 0)
```

## 2. Блок `try`

Блок `try` содержит код, в котором может возникнуть исключение.

Пример:

```python
try:
    print(10 / 0)
```

## 3. Блок `except`

Блок `except` выполняется при возникновении ошибки в блоке `try`.

Пример:

```python
try:
    print(10 / 0)
except:
    print("Ошибка")
```

## 4. Обработка конкретных исключений

Можно обрабатывать определённые типы ошибок отдельно.

Пример:

```python
try:
    number = int("abc")
except ValueError:
    print("Неверное значение")
```

## 5. Блок `else`

Блок `else` выполняется, если в блоке `try` не возникло исключений.

Пример:

```python
try:
    print(10 / 2)
except:
    print("Ошибка")
else:
    print("Успешно")
```

## 6. Блок `finally`

Блок `finally` выполняется всегда, независимо от наличия ошибок.

Пример:

```python
try:
    print(10 / 2)
finally:
    print("Завершение программы")
```

## 7. Создание исключений

Исключения можно вызывать самостоятельно с помощью оператора `raise`.

Пример:

```python
age = -1

if age < 0:
    raise ValueError("Возраст не может быть отрицательным")
```

## 8. Практическое использование исключений

Обработка исключений делает программу более устойчивой к ошибкам пользователя и непредвиденным ситуациям.

Пример:

```python
try:
    age = int(input("Возраст: "))
except ValueError:
    print("Введите число")
```

# Exceptions в Python

## Введение

Исключения - это ошибки времени выполнения, которые возникают во время работы программы. Вместо мгновенного завершения Python позволяет перехватывать такие ошибки и обрабатывать их с помощью блоков `try`, `except`, `else` и `finally`. Это делает программы устойчивыми к ошибкам, связанным с вводом данных, файлами, сетью и другими внешними факторами.

## Базовая обработка исключений

Обработка исключений начинается с блока `try`, внутри которого размещается код, способный вызвать ошибку. Если ошибка возникает, выполнение передается в блок `except`.

Пример:

```python
try:
    number = int(input("Введите число: "))
    print("Введенное число:", number)
except ValueError:
    print("Преобразование прошло неудачно")
print("Завершение программы")
```

В данном случае обрабатывается исключение `ValueError`, которое возникает при некорректном преобразовании строки в число.

## Основные типы исключений

В Python существует иерархия исключений. Базовыми являются:

- `BaseException` - корневой класс всех исключений

- `Exception` - базовый класс для большинства обрабатываемых ошибок

- `ArithmeticError` - ошибки арифметических операций

- `BufferError` - ошибки работы с буфером

- `LookupError` - ошибки доступа к элементам коллекций


От этих классов наследуются конкретные исключения:

- `IndexError` - выход индекса за пределы списка

- `KeyError` - отсутствие ключа в словаре

- `OverflowError` - переполнение числового типа

- `RecursionError` - превышение глубины рекурсии

- `TypeError` - операция с несовместимыми типами

- `ValueError` - корректный тип, но некорректное значение

- `ZeroDivisionError` - деление на ноль

- `NotImplementedError` - не реализованный метод

- `ModuleNotFoundError` - модуль не найден при импорте

- `OSError` - ошибки операционной системы

## Обработка нескольких исключений

Можно обрабатывать разные типы ошибок отдельно, добавляя несколько блоков `except`. Python выбирает первый подходящий блок.

Пример:

```python
try:
    number1 = int(input("Введите первое число: "))
    number2 = int(input("Введите второе число: "))
    print("Результат деления:", number1 / number2)
except ValueError:
    print("Некорректный ввод")
except ZeroDivisionError:
    print("Деление на ноль")
except BaseException:
    print("Общая ошибка")
print("Завершение программы")
```

Если отсутствует подходящий блок `except`, программа завершится с ошибкой.

Также можно обрабатывать несколько исключений в одном блоке:

```python
try:
    number1 = int(input("Введите первое число: "))
    number2 = int(input("Введите второе число: "))
    print(number1 / number2)
except (ValueError, ZeroDivisionError):
    print("Ошибка ввода или деление на ноль")
```

## Получение информации об исключении

С помощью `as` можно получить объект исключения и вывести его сообщение.

```python
try:
    number = int(input("Введите число: "))
except ValueError as e:
    print("Ошибка:", e)
```

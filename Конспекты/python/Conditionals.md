#Conditionals #Python #if #else #elif
## 1. Оператор `if`

Оператор `if` используется для выполнения кода при выполнении заданного условия.

Пример:

```python
age = 18

if age >= 18:
    print("Доступ разрешён")
```

## 2. Оператор `else`

Оператор `else` выполняется, если условие в `if` оказалось ложным.

Пример:

```python
age = 16

if age >= 18:
 print("Доступ разрешён")
else:
  print("Доступ запрещён")
```

## 3. Оператор `elif`

Оператор `elif` позволяет проверить дополнительное условие, если предыдущее оказалось ложным.

Пример:

```python
score = 75

if score >= 90:
  print("Отлично")
elif score >= 70:
  print("Хорошо")
```

## 4. Вложенные условия

Условные конструкции можно размещать внутри других условий для создания более сложной логики.

Пример:

```python
age = 20

if age >= 18:
   if age >= 21:
        print("Можно всё")
```

## 5. Логические операторы в условиях

В условиях можно использовать операторы `and`, `or` и `not` для объединения нескольких проверок.

Пример:

```python
age = 20

if age >= 18 and age < 65:
    print("Подходит")
```

## 6. Сравнение значений

Условия часто используют операторы сравнения для проверки данных.

Пример:

```python
x = 10

if x == 10:
    print("Равно десяти")
```

## 7. Тернарный оператор

Тернарный оператор позволяет записать простое условие в одну строку.

Пример:

```python
age = 20

status = "Взрослый" if age >= 18 else "Ребёнок"
```

## 8. Практическое использование условий

Условные конструкции применяются для принятия решений и управления выполнением программы.

Пример:

```python
password = "1234"

if password == "1234":
    print("Вход выполнен")
```

### 1. Создание строк

Строки используются для хранения текстовой информации. Они создаются с помощью одинарных или двойных кавычек.

Пример:

```python
name = "Alex"
```

### 2. Индексация строк

Каждый символ строки имеет свой индекс. Индексация позволяет получать доступ к отдельным символам строки.

Пример:

```python
word = "Python"
print(word[0])
```

### 3. Срезы строк

Срезы используются для получения части строки по диапазону индексов.

Пример:

```python
word = "Python"
print(word[0:3])
```

### 4. Объединение строк

Строки можно объединять с помощью оператора `+`.

Пример:

```python
first = "Hello"
second = "World"
print(first + " " + second)
```

### 5. Повторение строк

Оператор `*` позволяет повторять строку несколько раз.

Пример:

```python
text = "Hi "
print(text * 3)
```

### 6. Основные методы строк

Методы позволяют изменять и обрабатывать текст.

Пример:

```python
text = "python"
print(text.upper())
```

### 7. Форматирование строк

Форматирование используется для вставки значений переменных в текст.

Пример:

```python
name = "Alex"
print(f"Привет, {name}")
```

### 8. Проверка содержимого строк

Методы проверки позволяют определить свойства строки.

Пример:

```python
print("123".isdigit())
```

### 9. Длина строки

Функция `len()` определяет количество символов в строке.

Пример:

```python
text = "Python"
print(len(text))
```

### 10. Работа со строковыми данными

Строки используются для хранения, обработки и вывода текстовой информации.

Пример:

```python
name = input("Введите имя: ")
print(name)
```

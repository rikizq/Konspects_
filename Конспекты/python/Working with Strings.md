### Введение

Строка (`str`) - это тип данных, предназначенный для хранения текста. Строки представляют собой последовательность символов и создаются с помощью одинарных, двойных или тройных кавычек. После создания содержимое строки нельзя изменить, поэтому строки относятся к неизменяемым объектам.

### Индексация строк

Каждый символ строки имеет собственный индекс. Нумерация начинается с нуля. Также можно использовать отрицательные индексы для доступа к символам с конца строки.

```python
text = "Excuse me, sir"

print(text[0])   # Результат: E (первый символ)
print(text[-1])  # Результат: r (последний символ)
```

### Срезы строк

Срезы позволяют получить часть строки. При создании среза можно указать начальный индекс, конечный индекс и шаг. Правая граница в результат не включается.

```python
text = "Excuse me, sir"

print(text[0:6])   # Результат: Excuse
print(text[::-1])  # Результат: ris ,em esucxE
```

### Объединение и повторение

Строки можно объединять оператором `+`. Для повторения строки несколько раз используется оператор `*`.

```python
print("Excuse me, " + "sir")

print("Oi! " * 3)  # Результат: Oi! Oi! Oi!
```

### Форматирование строк

Для вставки значений переменных внутрь текста удобно использовать f-строки. Такой способ делает код более читаемым и позволяет вставлять выражения прямо в строку.

```python
name = "Butcher"

print(f"Excuse me, sir. Меня зовут {name}.")
```

### Полезные методы строк

Python предоставляет большое количество методов для обработки текста. Метод `upper()` переводит символы в верхний регистр, `lower()` - в нижний, `replace()` заменяет часть текста, а `strip()` удаляет лишние пробелы по краям строки.

```python
text = "  excuse me, sir  "

print(text.upper())                  # Результат: EXCUSE ME, SIR
print(text.strip())                  # Результат: excuse me, sir
print(text.replace("sir", "mate"))   # Результат: excuse me, mate
```

### Разделение и объединение строк

Метод `split()` разбивает строку на список элементов по указанному разделителю. Метод `join()` выполняет обратную операцию и объединяет элементы списка в одну строку.

```python
words = "Excuse,me,sir".split(",")

print(words)
# Результат: ['Excuse', 'me', 'sir']

text = " ".join(words)

print(text)
# Результат: Excuse me sir
```

### Проверка содержимого

Некоторые методы позволяют проверять содержимое строки. Например, `isdigit()` определяет, состоит ли строка только из цифр.

```python
print("123".isdigit())      # Результат: True
print("Butcher".isalpha())  # Результат: True
print("Oi123".isdigit())    # Результат: False
```

### Длина строки

Количество символов в строке можно узнать с помощью функции `len()`.

```python
text = "Excuse me, sir"

print(len(text))  # Результат: 14
```

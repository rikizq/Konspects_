## 1. Цикл `for`

Цикл `for` используется для перебора элементов последовательности или выполнения кода определённое количество раз.

Пример:

```python
for i in range(5):
    print(i)
```

## 2. Цикл `while`

Цикл `while` выполняется до тех пор, пока условие остаётся истинным.

Пример:

```python
count = 0

while count < 5:
    print(count)
    count += 1
```

## 3. Функция `range()`

Функция `range()` создаёт последовательность чисел, которая часто используется в цикле `for`.

Пример:

```python
for i in range(1, 6):
    print(i)
```

## 4. Оператор `break`

Оператор `break` полностью прекращает выполнение цикла.

Пример:

```python
for i in range(10):
    if i == 5:
        break
    print(i)
```

## 5. Оператор `continue`

Оператор `continue` пропускает текущую итерацию и переходит к следующей.

Пример:

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

## 6. Вложенные циклы

Один цикл может находиться внутри другого цикла. Такие конструкции используются для работы с таблицами и двумерными данными.

Пример:

```python
for i in range(3):
    for j in range(2):
        print(i, j)
```

## 7. Перебор строк

Циклы позволяют обрабатывать каждый символ строки по отдельности.

Пример:

```python
word = "Python"

for letter in word:
    print(letter)
```

## 8. Практическое использование циклов

Циклы применяются для автоматизации повторяющихся действий и обработки больших объёмов данных.

Пример:

```python
numbers = [1, 2, 3]

for number in numbers:
    print(number)
```

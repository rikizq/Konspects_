#Functions #Built-in #Функции #def #Аргументы

## 1. Что такое функции

Функции используются для группировки кода в отдельные блоки, которые можно вызывать многократно. Это помогает избегать повторения кода и делает программу более удобной для поддержки.

Пример:

```python
def greet():
    print("Hello")

greet()
```

## 2. Создание функций

Функции создаются с помощью ключевого слова `def`, после которого указывается имя функции и круглые скобки.

Пример:

```python
def say_hello():
    print("Hello World")
```

## 3. Параметры и аргументы

Функции могут принимать данные через параметры. Значения, передаваемые при вызове функции, называются аргументами.

Пример:

```python
def greet(name):
    print("Hello", name)

greet("Alex")
```

## 4. Возврат значения

Оператор `return` используется для возврата результата работы функции.

Пример:

```python
def add(a, b):
    return a + b

print(add(5, 3))
```

## 5. Область видимости переменных

Переменные, созданные внутри функции, доступны только внутри неё. Такие переменные называются локальными.

Пример:

```python
def test():
    x = 10
    print(x)
```

## 6. Встроенная функция `print()`

Функция `print()` выводит данные на экран.

Пример:

```python
print("Python")
```

## 7. Встроенная функция `input()`

Функция `input()` получает данные от пользователя.

Пример:

```python
name = input("Введите имя: ")
```

## 8. Встроенная функция `len()`

Функция `len()` возвращает количество элементов в объекте или длину строки.

Пример:

```python
text = "Python"
print(len(text))
```

## 9. Встроенные функции преобразования типов

Функции `int()`, `float()`, `str()` и `bool()` используются для преобразования типов данных.

Пример:

```python
number = int("10")
```

## 10. Другие встроенные функции

Python содержит множество встроенных функций для работы с данными и вычислениями, например `max()`, `min()`, `sum()` и `type()`.

Пример:

```python
numbers = [1, 2, 3]
print(max(numbers))
```

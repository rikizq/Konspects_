#Docker #Деплой #Kubernetes #Хостинг

Деплой - процесс развертывания приложения на хостинге, для обеспечения внешнего доступа.

Docker - инструмент для контейнеризации, который упаковывает приложение с зависимостями в легкий изолированный контейнер. Он изолирует процессы на уровне ядра и работает на одном хосте. Его задача увеличить совместимость, упростить деплой и воспроизводимость.

Kubernetes - инструмент для управления контейнерами, обеспечивающий высокую эффективность и отказоустойчивость развернутых систем.

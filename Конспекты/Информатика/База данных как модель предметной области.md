#База_данных #БД #Предметная_область #Типы #17/02 
### Определения
База данных (БД) - совокупность данных организованных по определенным правилам, отражающие состояние объектов и их отношений в некоторых предметной области, предназначенная для хранения постоянной памяти и ее постоянного применения.

Предметная область - это целевая тема конкретного проекта. В свою очередь доменная или концептуальная модель призвана выразить значение терминов и понятий, а также установление верных взаимосвязей между различными понятиями.
## Введение
Базу данных можно рассматривать как информационную модель предметной области.

### Типы баз данных
1. Реляционная база данных - это тип БД, в которой данные организованы в виде связанных таблиц.
2. Сетевые базы данных - в сетевых базах данных между таблицами и записями может быть несколько различных связей, в том числе несколько предков. Особенности сетевой базы заключается в том, что в ней запоминаются все связи и все содержимое для каждых из них. Иначе говоря поиск в такой базе данных индексируется.
3. Иерархические базы данных - это модель организующая в древовидную структуру, где каждый дочерний элемент подчиняется только одному родительскому.

Базы данных помогают хранить и упорядочивать информацию о предметной области. Выбор типа базы зависит от структуры данных и целей работы с ними. Правильная организация БД делает работу с информацией удобной и эффективной.

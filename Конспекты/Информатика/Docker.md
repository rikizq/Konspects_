#Docker #Image #Container #Dockerfile #DockerHub 
### Основное определение
Docker - это платформа для упаковки, доставки и запуска приложений в контейнерах.
### Ключевые элементы
- Image (образ) - шаблон для контейнера
- Container (контейнер) - запущенный экземпляр образа
- Dockerfile - файл с инструкциями для создания образа
- Docker Hub - публичный реестр образов
### Основные команды
`docker build` - создать образ.
`docker run` - запустить контейнер.
`docker ps` - показать работающие контейнеры.
`docker stop` / `docker rm` - остановка и удаление контейнера.
 
### Преимущества Docker
1. Консистентность окружений
2. Изоляция приложений
3. Лёгковесность по сравнению с виртуальными машинами
4. Удобство для микросервисов и CI/CD

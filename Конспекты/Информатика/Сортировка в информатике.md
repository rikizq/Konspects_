### Сортировка пузырьком (Bubble Sort)
Алгоритм последовательно проходит по массиву и сравнивает соседние элементы. Если элемент слева больше элемента справа, они меняются местами.
После каждого прохода самый большой элемент "всплывает" в конец массива (отсюда и название - "пузырёк"). Проходы повторяются, пока весь массив не станет отсортированным.

Принцип работы:
1. Сравниваются два соседних элемента.
2. Если первый больше второго, они меняются местами.
3. Процесс повторяется для всех соседних пар до конца массива (после первого прохода максимальный элемент окажется в конце).
4. Массив перебирается снова (исключая последний, уже отсортированный элемент), пока не потребуется перестановки.

### Сортировка выбором (Selection Sort)
Простой алгоритм, который делит массив на отсортированную (в начале) и неотсортированную части. На каждом шаге он находит минимальный элемент в неотсортированной части и меняет его местами с первым неотсортированным элементом, постепенно упорядочивая массив. Сложность алгоритма составляет, что делает его неэффективным для больших списков. 

Принцип работы:
1. Найти минимальный элемент в текущем неотсортированном подмассиве.
2. Обменять этот минимальный элемент с первым элементом неотсортированной части.
3. Сдвинуть границу отсортированной части на один элемент вправо.
4. Повторять, пока все элементы не будут отсортированы.

### Сортировка вставками (Insertion Sort)
Простой алгоритм, который формирует отсортированный массив по одному элементу. Каждый следующий элемент из неотсортированной части вставляется в правильную позицию среди уже упорядоченных элементов.
Алгоритм эффективен для небольших массивов и частично отсортированных данных.

Принцип работы:
1. Первый элемент считается уже отсортированным.
2. Берется следующий элемент из неотсортированной части.
3. Он сравнивается с элементами в отсортированной части справа налево.
4. Элементы, которые больше текущего, сдвигаются вправо.
5. Текущий элемент вставляется на освободившееся место.
6. Процесс повторяется, пока все элементы не будут отсортированы.

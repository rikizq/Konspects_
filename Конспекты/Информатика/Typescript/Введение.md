```typescript
let words: string[] = ["Да.", "Нет.", "Возможно."];
let words_len = words.length;
let answer_index: number = Math.floor(Math.random() * words_len);

prompt("Задайте вопрос!")
console.log(words, words_len, answer_index)
alert(words[answer_index])
```

## TypeScript представляет язык программирования на основе JavaScript.

### Переменные и константы 

##### Переменные

Переменные — это именованные контейнеры для данных, значения которых могут изменяться в процессе выполнения программы. В TypeScript для объявления переменных используются два ключевых слова:

- `var` — традиционный способ объявления переменных, унаследованный из JavaScript

- `let` — современный способ объявления переменных, введенный в ES2015

##### Константы

Константы — это именованные значения, которые устанавливаются один раз и не могут быть изменены в дальнейшем. Для их объявления используется ключевое слово `const`

Понимание различий между `var`, `let` и `const` имеет решающее значение для написания надежного и поддерживаемого кода на TypeScript. Каждый из этих подходов имеет свои особенности в отношении области видимости, повторного объявления и времени инициализации, что напрямую влияет на качество и безопасность вашей программы.

### Типы данных

#### Преимущества статической типизации

1. Раннее обнаружение ошибок — компилятор TypeScript проверяет типы на этапе компиляции, выявляя несоответствия типов до запуска программы.

2. Улучшенная поддержка IDE — интегрированные среды разработки могут предоставлять умные подсказки, авто дополнение кода и навигацию по коду на основе информации о типах.

3. Самостоятельная документация — типы служат документацией, делая код более понятным и предсказуемым для разработчиков.

4. Безопасность рефакторинга — система типов помогает безопасно изменять код, предупреждая о местах, где изменения могут нарушить типовую согласованность.
## Основные категории типов

TypeScript поддерживает богатую систему типов, включающую:

- Примитивные типы — базовые типы данных: `boolean`, `number`, `string`, `bigint`

- Специальные типы — `any`, `unknown`, `never`, `void`, `null`, `undefined`

- Составные типы — массивы, кортежи, объекты, перечисления

- Продвинутые типы — объединения, пересечения, обобщенные типы (дженерики)

# Формула
Формула Бэкуса-Наура (сокр. БНФ.) - простой способ описывать правила синтаксиса языков программирования или других формальных систем. БНФ помогает точно и кратко объяснить как строить "предложения" (например, код программы из определенных "слов"). Также БНФ определяется как формальный способ записи синтаксических единиц, при котором они определяются через другие единицы или через элементарные символы.

## Алфавит Синтаксических единиц
1. Мета символ - ":: = " - разделяет левую часть правила (не терминал) от правой (определение), указывает что не терминал заменяется на указанную конструкцию.
2. (Черта) - " | " - (или альтернатива) указывает выбор между вариантами правой части правила, позволяет описывать несколько возможных форм.
3. Мета символ угловых скобок - "< >" (угловые скобки для не терминалов), служат для обрамления не терминалов.
### Эти три символа являются основой оригинальной БНФ.

## Термины БНФ
1. Правила (**Rule**) - формальное утверждение в БНФ, записываемое как "<не терминал :: =  Определение>" которое описывает как не терминал может быть заменен на последовательность символов.
2. Определение (**Definition**) - правая часть правила БНФ, состоящая из терминалов, не терминалов и других конструкций.
3. Терминал (**Terminal**) - конечный символ в БНФ, который не может быть далее разложен и представляет конкретные элементы языка программирования и представляет такие как ключевые слова, операторы или литералы.
4. Не терминал (**Non Terminal**) - символ в БНФ, представляющий категорию и абстрактную конструкцию языка, которая может быть заменена согласно правилам.

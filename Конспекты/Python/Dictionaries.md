**Dictionary** - встроенная структура данных Python для хранения информации в формате ключ → значение (_key-value_).

## Основные свойства

• Изменяемые (mutable), поэтому данные можно добавлять, изменять и удалять.
• Ключи должны быть уникальными.
• Ключи должны быть hashable. Обычно используются строки, числа и кортежи.
• Обеспечивают быстрый доступ к данным благодаря использованию хеш-таблиц.
• Сохраняют порядок добавления элементов (начиная с Python 3.7).

## Создание словарей

Словари создаются с помощью фигурных скобок `{}` или функции `dict()`.
Также можно создать пустой словарь и заполнить его позже.

## Основные операции

• Получение значения по ключу.
• Безопасное получение значения через метод `get()`.
• Добавление новых элементов.
• Изменение существующих значений.
• Удаление элементов.

## Полезные методы

• `keys()` возвращает все ключи.
• `values()` возвращает все значения.
• `items()` возвращает пары ключ–значение.

## Перебор словаря

Можно перебирать:
• ключи;
• значения;
• ключи и значения одновременно.
Для проверки существования ключа используется оператор `in`.

## Dictionary Comprehension
Позволяет создавать словари в компактной форме на основе циклов и условий.

## Вложенные словари
Словари могут содержать другие словари, что позволяет хранить более сложные структуры данных.

## Где используются словари

• JSON-данные.
• API-запросы и ответы.
• Конфигурационные файлы и настройки.
• Обработка и анализ данных.
• Подсчёт и группировка информации.

# HashMap в Python: основные принципы и применение

HashMap (хэш-карта) - одна из ключевых структур данных в программировании. В Python её реализацией является встроенный словарь `dict`, который используется практически во всех приложениях: от простых скриптов до крупных веб-сервисов.
## Что такое HashMap

HashMap хранит данные в формате:

```python
ключ -> значение
```

Каждый ключ связан с определённым значением, а доступ к данным происходит очень быстро благодаря механизму хеширования.

Пример словаря в Python:

```python
users = {
    "user_1": "Alice",
    "user_2": "Bob"
}
```
## Как работает HashMap

При обращении к элементу:

```python
users["user_1"]
```

Python:

1. вычисляет хэш ключа;
2. определяет индекс хранения;
3. получает значение напрямую из памяти.

В отличие от списка, где поиск может происходить последовательно, hash map обеспечивает почти мгновенный доступ к данным.

Средняя сложность операций:
- поиск - `O(1)`
- добавление - `O(1)`
- удаление - `O(1)`
## Хэш-функция

Хэш-функция преобразует ключ в числовое значение - hash.

Одинаковые входные данные всегда создают одинаковый хэш, что позволяет быстро находить нужное значение.

В Python используется встроенная функция:

```python
hash("hello")
```
## Коллизии

Коллизия возникает, когда разные ключи получают одинаковый хэш.

Python автоматически решает эту проблему с помощью специальных механизмов:
- probing (поиск следующего свободного места);
- перераспределение таблицы при увеличении количества данных.
## Основные операции со словарями

### Создание

```python
inventory = {
    "apple": 10,
    "banana": 5
}
```
### Добавление

```python
inventory["orange"] = 7
```
### Обновление

```python
inventory["apple"] = 15
```
### Получение значения

```python
inventory["banana"]
```
### Безопасный доступ

```python
inventory.get("grape")
```

Метод `get()` предотвращает ошибку `KeyError`, если ключ отсутствует.
### Удаление

```python
del inventory["banana"]
```
## Важные особенности HashMap

### Ключи должны быть хэшируемыми

Разрешено:
- строки
- числа
- tuple
Нельзя использовать:
- list
- dict
## Полезные структуры из collections

### defaultdict

Автоматически создает значение по умолчанию.

```python
from collections import defaultdict
```

Используется для группировки данных.
### Counter

Позволяет быстро подсчитывать количество элементов.

```python
from collections import Counter
```
### ChainMap

Объединяет несколько словарей в одну структуру.

```python
from collections import ChainMap
```

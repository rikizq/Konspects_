## Из чего состоит ПК?

Компьютер - это комплекс периферийных и вычислительных устройств, предназначенных для конечных задач пользователя.

## Устройство системного блока (СБ)

Среднестатистический СБ состоит из:

1. Материнская плата (МП) Определяется набором поддерживаемых устройств, макс. вычислительной мощностью, назначением. Основную функциональность МП определяет чипсет (кол-во и скорость интерфейсов подключения, скорость обмена данными между компонентами МП).
    
2. Устройства Ввода/Вывода (I/O interface)
    
    - На СБ есть отдельный блок с набором портов для подключения различных устройств (I/O interface).
3. CPU (Центральный процессор)
    
    - Процессор нужен для вычисления сложных данных.
    - Каждое ядро имеет сложную структуру.
    - Транзисторы имеют очень маленький размер (12 нанометров).
    - Сильно взаимосвязан с чипсетом и ОЗУ.
4. Накопители данных
    
    - Существует несколько типов накопителей, напр. HDD, SSD, NVMe.
    - Они различаются скоростью обработки, технологией записи и интерфейсом.
    - HDD - медленнее, чем остальные типы накопителей.
    - Данные записываются на "болванку" (для оптических дисков, ?).

## Основные интерфейсы ПК

### Интерфейсы компонентов

- Разъём ОЗУ (димм).
- Разъём PCI-Express x16(для видео-, звуковых карт).
- Разъём PCI-E x1-4 (менее производительный).
- Разъёмы SATA 3-4 (для подключения HDD и SSD).
- Разъём M.2 (для подкл. SSD SATA или NVMe).
- Разъёмы USB (для подкл. USB-портов на корпусе или задней панели).
- Разъём Audio (для подкл. портов Audio на корпусе).
- Разъёмы питания МП (для питания процессора и МП).

### Интерфейсы периферийных устройств

На среднестатистической МП есть отдельный блок с набором портов для подключения различных устройств (I/O interface).

- CPU Процессор нужен для вычисления сложных данных. Каждое ядро имеет сложную структуру. Транзисторы имеют очень маленький размер (12 нанометров). Сильно взаимосвязан с чипсетом и ОЗУ.
- Накопители данных Существует несколько типов накопителей, напр. HDD, SSD, NVMe. Они различаются скоростью обработки информации, технологией записи и интерфейсом подключения. HDD - медленнее, чем остальные типы накопителей. Данные записываются на "болванку".

```typescript
function inorderTraversal(root: TreeNode | null): number[] {
    const result: number[] = []
    const stack: TreeNode[] = []
    let current: TreeNode | null = root

    while (current !== null || stack.length > 0) {
        // Идём максимально влево
        while (current !== null) {
            stack.push(current)
            current = current.left
        }

        // Обрабатываем узел
        current = stack.pop()!
        result.push(current.val)

        // Переходим в правое поддерево
        current = current.right
    }

    return result
}
```

### Объяснение
Этот код реализует inorder-обход бинарного дерева без рекурсии, используя стек. Функция принимает корень бинарного дерева root и возвращает массив значений узлов в порядке inorder-обхода. Создаются:
result - массив для хранения результата обхода,
stack - стек для сохранения узлов при спуске в дерево,
current - указатель на текущий узел, изначально указывает на корень.
Алгоритм работает в цикле, пока существует текущий узел или в стеке есть элементы.
Сначала происходит спуск как можно левее: каждый текущий узел добавляется в стек, после чего переход идёт к его левому потомку. Это повторяется до тех пор, пока левый узел отсутствует. Когда дальнейший спуск влево невозможен, из стека извлекается последний узел - это следующий узел в inorder-порядке. Его значение добавляется в result.
После обработки узла происходит переход в его правое поддерево, и алгоритм повторяется по тем же правилам. В итоге функция возвращает массив значений всех узлов дерева, упорядоченных согласно inorder-обходу.

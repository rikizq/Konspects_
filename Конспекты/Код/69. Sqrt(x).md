```python
function mySqrt(x: number): number {
    if (x < 2) return x;
    
    let left = 0;
    let right = x;
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        const square = mid * mid;
        
        if (square === x) {
            return mid;
        } else if (square < x) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return right;
}

function mySqrt2(x: number): number {
    if (x < 2) return x;
    
    let left = 1;
    let right = Math.floor(x / 2);
    
    while (left <= right) {
        const mid = Math.floor((left + right) / 2);
        const square = mid * mid;
        
        if (square === x) {
            return mid;
        } else if (square < x) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }
    
    return right;
}
```

### Объяснение
Этот код содержит две функции mySqrt и mySqrt2, которые вычисляют целую часть квадратного корня из неотрицательного числа x, то есть находят наибольшее целое число, квадрат которого не превышает x. Обе функции используют алгоритм бинарного поиска: если x меньше 2, сразу возвращается само число, так как корень из 0 равен 0, а из 1 равен 1. В первой функции поиск ведётся в диапазоне от 0 до x, во второй — в более узком диапазоне от 1 до x / 2, что делает её немного эффективнее. В цикле выбирается середина текущего диапазона, вычисляется квадрат этого числа и сравнивается с x: если квадрат равен x, функция сразу возвращает найденное число, если квадрат меньше x, поиск продолжается в правой половине диапазона, а если больше — в левой. Когда подходящее значение не найдено точно, цикл завершается, и функция возвращает значение right, которое в этот момент содержит искомую целую часть квадратного корня.

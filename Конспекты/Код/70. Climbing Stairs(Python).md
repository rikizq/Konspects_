```python
class Solution:
    def climbStairs(self, n: int) -> int:
        if n <= 2:
            return n
        
        prev2, prev1 = 1, 2
        for _ in range(3, n + 1):
            prev2, prev1 = prev1, prev1 + prev2
        
        return prev1

```

Задача решается с помощью динамического программирования. Количество способов подняться на n ступеней равно сумме способов для n-1 и n-2, так как на каждую ступень можно попасть либо шагом на 1, либо на 2. В коде используются только две переменные для хранения предыдущих значений, что позволяет сократить использование памяти до O(1)

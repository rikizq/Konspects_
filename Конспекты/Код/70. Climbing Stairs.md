```python
function climbStairs(n: number): number {
    if (n <= 2) return n;
    
    // Используем переменные для хранения предыдущих двух значений
    let prev1 = 1;  // f(n-2)
    let prev2 = 2;  // f(n-1)
    
    for (let i = 3; i <= n; i++) {
        // Текущее значение - сумма двух предыдущих
        const current = prev1 + prev2;
        
        // Обновляем значения для следующей итерации
        prev1 = prev2;
        prev2 = current;
    }
    
    return prev2;
}
```

### Объяснение
Этот код реализует функцию climbStairs, которая считает, сколькими разными способами можно подняться на лестницу из n ступенек, если за один шаг можно подняться либо на одну, либо на две ступеньки. Если n меньше или равно 2, функция сразу возвращает n, потому что на одну ступеньку есть один способ, а на две — два способа. Далее используется итеративный подход, похожий на вычисление чисел Фибоначчи: переменные prev1 и prev2 хранят количество способов для двух предыдущих значений — для n - 2 и n - 1. В цикле от 3 до n на каждой итерации вычисляется текущее количество способов как сумма двух предыдущих, после чего значения обновляются для следующего шага. В конце функция возвращает prev2, в котором хранится количество способов добраться до n-й ступеньки.

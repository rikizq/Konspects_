#Код #Исключения #Python 
```python
try:
    number = 3
    if number > 50 or number < 3:
        raise Exception("Неккоректный возраст")
    print(number)
except ZeroDivisionError:
    print("возникает при делении на ноль")
except ValueError:
    print("Преобразование прошло неудачно")
except Exception as e:
    print(e)
finally:
    print("Блок try завершил выполнение")
print("Завершение программы")
```

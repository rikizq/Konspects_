```python
class ListNode:
    # Конструктор узла связного списка
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class Solution:
    def deleteDuplicates(self, head: ListNode) -> ListNode:
        current = head
        
        # Пока текущий элемент и следующий существуют
        while current and current.next:
            if current.val == current.next.val:
                # пропускаем дубликат
                current.next = current.next.next
            else:
                current = current.next
        
        return head
```

### Объяснение
Эта программа реализует алгоритм удаления дубликатов из отсортированного односвязного списка на языке Python. Для представления элементов списка используется класс ListNode, в котором хранятся значение узла и ссылка на следующий элемент.

Метод deleteDuplicates класса Solution принимает начальный узел списка и последовательно проходит по всем его элементам. В процессе работы происходит сравнение значений текущего узла и следующего за ним. Если значения совпадают, следующий узел исключается из списка путём изменения ссылки next. Если значения различаются, осуществляется переход к следующему элементу.

#leetcode #math #reverseinteger
### Описание
В данном коде требуется получить число, образованное обратным порядком цифр исходного целого числа, с сохранением знака и проверкой на выход за пределы допустимого диапазона.

```python
class Solution:
    def reverse(self, x: int) -> int:
        upper_limit = 2147483648 # Верхняя граница для 32-битного знакового целого числа (2^31)
        lower_limit = -2147483648 # Нижняя граница для 32-битного знакового целого числа (-2^31)
        reversed_x = 0 # Переменная для хранения перевёрнутого числа

        if x < 0:
        # Условный оператор: проверяем, отрицательное ли число
        
            reversed_x = int(str(abs(x))[::-1]) * -1
        else:
            reversed_x = int(str(abs(x))[::-1])
            # Переворачиваем цифры числа без изменения знака

        if reversed_x >= upper_limit or reversed_x < lower_limit:
        # Проверяем, выходит ли число за допустимые границы
	    # 32-битного знакового целого типа
        
            return 0
            # Если выходит — возвращаем 0
            
        else:
            return reversed_x
            # Возвращаем перевёрнутое число
```

### Определение ограничений:
Устанавливаются верхнее и нижнее ограничение для результата (границы целочисленного диапазона):  
Переворот числа:
    Если число положительное, оно сразу переворачивается путем перевода в строку, разворота символов и последующего преобразования обратно в число.
    Если число отрицательное, берется его модуль (абсолютное значение), переворачивается, а затем умножается на минус единицу, восстанавливая изначальный знак.
### Проверка переполнения
Итоговое перевернутое число сравнивается с установленными ограничениями. Если оно вышло за рамки диапазона, возвращается 0, иначе возвращается само перевернутое число.

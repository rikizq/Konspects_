#palindrome #leetcode
### Описание
В данном коде требуется определить, является ли целое число палиндромом, то есть одинаково ли оно читается в прямом и обратном порядке.

```python
class Solution:
    def isPalindrome(self, x: int) -> bool:
        return str(x) == str(x)[::-1]  # Преобразование числа в строку и проверка палиндрома
```

### Преобразование числа в строку (str(x))
Преобразуем целое число x в строковую форму, поскольку строки позволяют легко сравнивать символы слева направо и справа налево.
Обратная строка (str(x)[::-1]):Используем срез [::-1], чтобы получить обратную версию исходной строки.
### Проверка равенства 
Затем сравниваем оригинальную строку с её обратной версией. Если обе равны, значит, число является палиндромом.

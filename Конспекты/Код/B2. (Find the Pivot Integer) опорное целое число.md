#Pivor_Integer #leetcode
### Описание
В данном коде требуется определить опорное целое число, при котором сумма натуральных чисел от 1 до этого числа равна сумме чисел от него до n. Если такое число отсутствует, возвращается −1.

```python
class Solution:
    def pivotInteger(self, n: int) -> int:

        #Формула Гаусса для нахождения общей суммы
        res0 = (n * (n + 1) // 2)

        #находим квадртаный корень от общей суммы
        res1 = int(res0**0.5)

        #если произведение корней равно общей сумме, вернуть корень
        if res1 * res1 == res0:
            return res1

        #если условие не сработает, вернется -1
        return -1
```

Код реализует быстрый способ нахождения опорного числа x для заданного n. Суть метода заключается в следующем:

Считаем полную сумму чисел от 1 до n по формуле Гаусса.

Эта сумма проверяется на предмет того является ли она полным квадратом. Если да то её квадратный корень и будет искомым опорным числом.

Если сумма точный квадрат выводим её корень как опорное число, иначе возвращаем −1.

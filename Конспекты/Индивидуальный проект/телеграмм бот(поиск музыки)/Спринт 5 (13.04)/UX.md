
### 1) Два языка

Все сообщения сделаны в формате:
Русский (English)
Например:
Ничего не найдено (Nothing found)
Это делает бота удобным как для русскоязычных, так и для англоязычных пользователей.

### 2) Удобный старт

Команда `/start` сразу показывает:
назначение бота
что можно отправить
список всех команд
подсказки по поиску

Пользователь сразу понимает, как пользоваться ботом.
### 3) Поиск без команд

Пользователь может просто отправить:
eminem
или
shape of you
и бот сразу начнет поиск.

Это делает использование быстрым и интуитивным.
### 4) Защита кнопок

Кнопки пагинации привязаны к ID пользователя.
Другой человек не сможет нажимать чужие кнопки.

Это улучшает:
- приватность
- удобство
- предсказуемость поведения
### 5) Минимальный интерфейс

Интерфейс специально сделан:
1. без лишних эмодзи 
2. без перегруженного текста
3. только нужные команды
4. только стрелки для навигации

Это улучшает читаемость и делает UX профессиональным.

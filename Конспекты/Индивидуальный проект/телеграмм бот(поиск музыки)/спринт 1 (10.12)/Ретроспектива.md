## 1. Выполненные задачи
    
Создан Telegram-бот с помощью библиотеки `aiogram`.
        
Бот умеет принимать текстовые сообщения от пользователя.
        
Реализован поиск музыки через публичный iTunes Search API по текстовому запросу (название трека или исполнитель).
        
Бот возвращает несколько результатов с названием трека, исполнителем и ссылкой на превью.
        
Настроен бот в @BotFather:

установлен токен

добавлено имя, описание и фотка

можно запускать бота и получать сообщения.

Оформление команд в @BotFather для удобства пользователя.

---

## 2. Невыполненные задачи

- Добавление кнопок навигации (например, «Следующий трек» / «Стоп») для просмотра нескольких результатов в интерактивном виде.
    
- Отправка аудио-превью напрямую через Telegram вместо ссылок.
    
- Расширение функционала: поиск по альбомам, жанрам или добавление истории запросов

## 3. Достигнутый результат

Да, основной результат достигнут:

Бот принимает текстовый запрос,

ищет музыку,

возвращает корректный результат пользователю.

Пользователь может использовать бота для быстрого поиска треков и прослушивания превью.

---

## 4. Что можно улучшить / исправить

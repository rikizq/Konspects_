#Алгоритмы #Виды #Характеристики #Переменные
**Алгоритмы** - это последовательность шагов или инструкций, необходимых для выполнения определённой задачи. Как и любая информация, алгоритм может быть представлен в различных формах - текст, описание, псевдокод, блок-схемы, программный код.

**Задача**: доставить пенал с 1 парты на последнюю

### Характеристики алгоритма

- Определённость Каждый алгоритм должен быть чётко определён и не вызывать двусмысленности.
- Конечность Алгоритм должен завершаться после конечного числа шагов.
- Входные и выходные данные
- Эффективность Алгоритм должен быть достаточно эффективен по времени и ресурсам.

### Виды алгоритмов

1. Линейный Каждый шаг выполняется последовательно (не содержит циклов или условий).
2. Разветвляющийся (с условиями) Использует условия для принятия решения, и в зависимости от них алгоритм может следовать разным путями.
3. Циклический Включает в себя повтор действий, которые выполняются до тех пор, пока не будет соблюдено определённое условие.

## Новое в Python

### Переменные

**Переменная** - именованная область памяти #ПК, в которой хранятся какие-либо данные. Значение переменной называется литералом и может представлять несколько типов данных. #Компьютер

Чтобы объявить переменную в Python, используется оператор присваивания (=). В левой части - имя переменной, в правой - её литерал. #Python

### Типы данных

Существуют простые типы данных в Python:

- Целые числа (int/integer): 2025 #int
- Вещественные числа (float): 22.3 #float
- Логический тип (bool/boolean): True, False #bool #True #False
- Строковый тип (str/string): "hey!" (пример строки) #str

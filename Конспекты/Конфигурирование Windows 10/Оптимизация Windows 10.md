# 1
- Телеметрия
1. Отключаем все параметры конфиденциальности
2. Нажмите Win+R, введите `services.msc`
3. Найдите “Функциональные возможности для подключенных пользователей и телеметрия”
4. ПКМ → Свойства → Тип запуска: “Отключена”
5. Нажмите “Остановить”

- Удаление лишних приложений
1. “Установка и удаление приложений”
2. Удалить ненужный софт и предустановленные программы

- Автозагрузка
1. Отключение в меню автозагрузки (Диспетчер задач /Параметры)
2. Оставить только нужные программы

- Windows Update (Обновления Windows)
1. Проверить наличие обновлений
2. Устанавливать только стабильные обновления

- Защита от вирусов и угроз
1. Полная проверка через Защитник Windows
2. Проверка подозрительных файлов

- Отключение гибернации
1. Командная строка от администратора
2. `powercfg -h off`

- Очистка мусора
1. Win+R → `%temp%`
2. Удалить временные файлы
3. Использовать “Очистку диска” (`cleanmgr`)

- Проверка системных файлов
1. Командная строка от администратора
2. `sfc /scannow`

- Проверка диска
1. Командная строка от администратора
2. `chkdsk /f /r`

- Электропитание
1. Панель управления → Электропитание
2. Режим “Высокая производительность”

- Визуальные эффекты
1. Win+R → `sysdm.cpl`
2. Дополнительно → Быстродействие
3. “Обеспечить наилучшее быстродействие”

- Оптимизация SSD/HDD
1. Win+R → `dfrgui`
2. Оптимизация дисков
3. Не заполнять SSD полностью (я оставляю 15% памяти от всего диска)

- Драйверы
1. Обновить драйвер видеокарты и чипсета
2. Удалить старые драйверы

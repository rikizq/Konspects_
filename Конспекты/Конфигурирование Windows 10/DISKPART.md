## Утилита DISKPART
DISKPART - это консольная утилита Windows для управления дисками, разделами и томами. Работает напрямую с дисковой разметкой, поэтому требует аккуратности и прав администратора.
### Назначение
DISKPART используется для:
просмотра физических дисков и разделов;
создания, удаления и форматирования разделов;
назначения и изменения букв дисков;
очистки дисков (полное удаление разметки);
подготовки дисков к установке Windows;
работы с USB-накопителями и виртуальными дисками (VHD).

### Основные команды:

#### Просмотр информации
list disk — список физических дисков
list volume — список томов
list partition — список разделов на выбранном диске

#### Выбор объекта
select disk N — выбрать диск (N — номер диска)
select volume N — выбрать том
select partition N — выбрать раздел

#### Работа с дисками и разделами
clean — удалить все разделы и разметку с диска
create partition primary — создать основной раздел
delete partition — удалить выбранный раздел
extend — расширить выбранный том

#### Форматирование
format fs=ntfs quick — быстрое форматирование в NTFS
format fs=fat32 — форматирование в FAT32

#### Назначение букв
assign letter=E — назначить букву диска
remove letter=E — удалить букву диска

#### Атрибуты
attributes disk set readonly — сделать диск только для чтения
attributes disk clear readonly — снять защиту от записи

#### Завершение работы
exit — выход из DISKPART

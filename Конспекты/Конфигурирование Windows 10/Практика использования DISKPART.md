### Введение
DISKPART - это мощный консольный инструмент управления дисками, разделами и томами в операционных системах Windows. Она предоставляет возможности, которые часто выходят за рамки графической оснастки "Управление дисками": работа с виртуальными дисками , преобразование стилей разделов (MBR/GPT), управление динамическими томами, настройка атрибутов GPT, расширение и сжатие томов, создание копий и т.д.

### Цель практики
Диагностика и навигация - с помощью list disk, list partition, list volume учусь получать актуальную информацию о конфигурации накопителей, что необходимо для планирования любых изменений.

Создание и настройка разделов - команды create partition primary с указанием размера позволяют гибко распределять пространство, а форматирование (format fs=... quick) и назначение букв (assign) доводят раздел до состояния готовности к использованию.

Работа с разными файловыми системами - использование NTFS и exFAT даёт понимание, когда какая система уместна (NTFS - для системных и больших томов Windows, exFAT - для съёмных носителей и совместимости).

Очистка и преобразование дисков - команды clean и convert gpt имитируют подготовку нового диска или полную перестройку существующего, что важно при замене оборудования или смене стиля разделов (например, переход на UEFI).

Расширение томов - операция extend отрабатывает сценарий увеличения раздела за счёт свободного места без потери данных, что часто требуется в эксплуатации серверов и рабочих станций.

Безопасная практика без рисков - все действия выполняются в симулированной среде, что позволяет экспериментировать и закреплять навыки без угрозы повреждения реальных данных или загрузочных записей.
## Сценарий тренировки DISKPART

 Диск 0 (MBR, 120 ГБ):
- C: (NTFS, 50 ГБ, системный, активный)
- F: (FAT32, 30 ГБ, Log1, логический)
- G: (NTFS, 40 ГБ, Log2, логический)
Свободно: 0 Б

Диск 1 (GPT, 500 ГБ):
- D: (NTFS, 500 ГБ, Storage)
Свободно: 0 Б

Диск 2 (GPT, 250 ГБ):
- H: (NTFS, 150 ГБ, Data)
- I: (NTFS, 80 ГБ, Extra, скрыт атрибутом GPT)
Свободно: 20 ГБ

#### Сценарий использования DISKPART

1. Просмотрите список дисков.
`list disk`

2. Выберите диск 0.
`select disk 0`

3. Просмотрите список разделов на диске 0.
`list partition`

4. Создайте на диске 0 новый простой том размером 20 ГБ в свободной области.
`create partition primary size=20480`

5. Отформатируйте новый том в файловую систему NTFS.
`list partition`
`select partition 4`
`format fs=ntfs quick`

6. Назначьте тому букву.		
`assign`

7. Просмотрите список томов.
`list volume`

8. Выберите диск 1.
`select disk 1`

9. Очистите диск 1 (преобразуйте в GPT).
`clean`
`convert gpt`

10. Создайте на диске 1 новый простой том размером 50 ГБ.
`create partition primary size=51200`

11. Отформатируйте его быстро в exFAT, метка "USB".
`select partition 1`
`format fs=exfat quick`

12. Назначьте букву.
`assign`

13. Расширьте том E на диске 1 до 100 ГБ, используя оставшееся место.
`list volume`
`select volume 4`
`extend size=51200`

#### Основные команды DISKPART
1. list disk - показать все физические диски.
2. select disk N - выбрать диск номер N для дальнейших операций.
3. list partition - показать разделы на выбранном диске.
4. list volume - показать все тома (логические диски с буквами).
5. select volume {буква/номер} - выбрать том по букве или номеру.
6. clean - удалить все разделы с выбранного диска (быстрая очистка).
7. clean all - очистить диск с обнулением каждого сектора.
8. convert gpt - преобразовать выбранный диск в GPT.
9. convert mbr - преобразовать выбранный диск в MBR.
10. create partition primary [size=N] - создать основной раздел (опционально указать размер в МБ).
11. create partition extended [size=N] - создать расширенный раздел (для MBR).
12. create partition logical [size=N] - создать логический диск внутри расширенного.
13. format fs=NTFS|FAT32|exFAT label="метка" quick - отформатировать выбранный том с указанием ФС, метки, быстро.
14. assign letter=X - назначить выбранному тому букву X.
15. delete partition - удалить выбранный раздел.
16. shrink desired=N - сжать том на N мегабайт.
17. extend size=N - расширить том на N мегабайт (если есть место)
18. gpt attributes=0x4000000000000000 - установить атрибуты GPT (например, скрыть раздел).
19. gpt attributes=0x1000000000000000 - установить атрибут "только для чтения".
20. offline disk - перевести выбранный диск в автономный режим.
21. online disk - вернуть диск в онлайн.
22. exit - выйти из DISKPART.

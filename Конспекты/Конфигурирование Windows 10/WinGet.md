### Введение
Windows Package Manager (winget) - официальный менеджер пакетов от Microsoft для операционных систем Windows 10, Windows 11 и Windows Server 2025. Это инструмент командной строки, который позволяет искать, устанавливать, обновлять и удалять приложения, упрощая управление программным обеспечением на компьютере.
### Установка
WinGet предустановлен в Windows 11 и современных версиях Windows 10 как часть компонента App Installer, распространяемого через Microsoft Store. Для работы используется команда winget, которую можно запускать в Windows Terminal, Windows PowerShell или Windows Command Prompt.
#### Основные команды WinGet
-  Поиск программы - winget search vscode
-  Установка - winget install Microsoft.VisualStudioCode
-  Удаление - winget uninstall Microsoft.VisualStudioCode
-  Обновление всех программ - winget upgrade --all
#### Поддерживаемые форматы.
Утилита Windows Package Manager поддерживает установку программ из разных типов файлов: .exe, .msi, .msix, .appx и их bundle-варианты (.msixbundle, .appxbundle). Это позволяет устанавливать как классические программы Windows, так и современные приложения экосистемы Microsoft.
#### Пакетная установка и конфигурация.
WinGet позволяет автоматизировать установку сразу нескольких программ с помощью командной строки или .bat-скриптов. Также можно экспортировать список установленных приложений в файл и затем восстановить их на другом компьютере с помощью команды winget import. Это удобно для быстрой настройки новой системы или восстановления рабочей среды.
### Заключение.
WinGet - это удобный и мощный инструмент для Windows, который превращает установку и обновление программ в быстрый и автоматизированный процесс. Он полезен для разработчиков, ИТ-специалистов и продвинутых пользователей, позволяя экономить время, поддерживать одинаковое окружение на разных устройствах и легко обновлять или устанавливать сразу несколько приложений одной командой.

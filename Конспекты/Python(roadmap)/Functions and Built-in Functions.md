### Введение

Функции используются для объединения набора инструкций в отдельный именованный блок кода, который можно вызывать многократно из разных частей программы. Они позволяют избежать дублирования кода, делают программу более структурированной и упрощают поддержку проектов. В Python функции создаются с помощью ключевого слова `def`, после которого указывается имя функции, круглые скобки и двоеточие. Код внутри функции записывается с обязательным отступом и выполняется только после вызова функции.

### Создание функций и работа с параметрами

Для объявления функции используется ключевое слово `def`. После имени функции в круглых скобках могут быть указаны параметры - специальные переменные, через которые функция получает данные извне. Значения, передаваемые при вызове функции, называются аргументами. Благодаря параметрам одна и та же функция может работать с разными данными без необходимости создавать множество похожих функций. Аргументы могут передаваться как по позиции, когда порядок имеет значение, так и по имени параметра, что делает код более понятным и удобным для чтения.

### Возврат значений

Для передачи результата работы функции используется оператор `return`. После его выполнения работа функции немедленно завершается, а указанное значение возвращается в место вызова. Функция может возвращать данные любого типа: числа, строки, списки, словари и другие объекты Python. Если оператор `return` отсутствует, функция автоматически возвращает специальное значение `None`, обозначающее отсутствие результата.

### Область видимости переменных

Переменные, созданные внутри функции, называются локальными и существуют только во время выполнения этой функции. После завершения работы функции такие переменные становятся недоступными. Это позволяет изолировать данные и предотвращает случайное изменение переменных из других частей программы. Переменные, созданные вне функций, называются глобальными и доступны во всей программе, однако их чрезмерное использование может усложнить отладку и сопровождение кода.

```python
# Функция с параметром
def greet(name):
    print("Hello", name)

# Функция с возвратом результата
def add(a, b):
    return a + b

greet("Alex")

result = add(5, 3)
print(result)

text = "Python"
print(len(text))
```

### Введение

Кортеж (`tuple`) - это структура данных в Python, предназначенная для хранения набора значений в одном объекте. Кортежи сохраняют порядок элементов и поддерживают доступ по индексам, однако после создания их содержимое нельзя изменить. Благодаря этому они хорошо подходят для хранения постоянных данных.

### Создание кортежей

Для создания кортежей используются круглые скобки `()`. Внутри могут находиться элементы любых типов данных: числа, строки, логические значения и другие объекты Python.

```python
person = ("Alex", 17, True)
```

### Работа с элементами

Каждому элементу соответствует свой индекс. Первый элемент имеет индекс `0`, второй - `1` и так далее. Индексы позволяют получать нужные значения из кортежа.

```python
foods = ("pizza", "burger", "sushi")

print(foods[0])
```

### Получение части кортежа

С помощью срезов можно извлекать несколько элементов сразу. Новый кортеж формируется на основе указанного диапазона индексов.

```python
numbers = (1, 2, 3, 4, 5)

print(numbers[1:4])
```

### Неизменяемость

После создания кортеж остаётся неизменным. Нельзя заменить существующий элемент, добавить новый или удалить старый. Попытка выполнить подобные действия приведёт к ошибке.

```python
numbers = (1, 2, 3)

# numbers[0] = 10
```

### Перебор элементов

Кортежи поддерживают перебор через цикл `for`, что позволяет последовательно работать с каждым элементом.

```python
numbers = (1, 2, 3)

for item in numbers:
    print(item)
```

### Распаковка

Значения из кортежа можно распределить по нескольким переменным за одну операцию. Количество переменных должно совпадать с количеством элементов.

```python
user = ("Alex", 17)

name, age = user
```

### Полезные функции

При работе с кортежами часто используются встроенные функции. `len()` возвращает количество элементов, `max()` и `min()` находят максимальное и минимальное значение, а `sum()` вычисляет сумму чисел.

```python
numbers = (1, 2, 3)

print(len(numbers))
print(sum(numbers))
```

### Проверка наличия элемента

Оператор `in` позволяет быстро определить, содержится ли значение в кортеже.

```python
numbers = (1, 2, 3)

print(2 in numbers)
```

### Практическое применение

Кортежи часто используют для хранения координат, настроек программы, данных, возвращаемых функциями, и другой информации, которая должна оставаться неизменной во время выполнения программы.

```python
coordinates = (10, 20)

x, y = coordinates

print(f"X: {x}")
print(f"Y: {y}")
```

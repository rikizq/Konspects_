### Введение

Преобразование типов (Type Casting) - это изменение одного типа данных на другой. В Python преобразование может происходить автоматически или выполняться программистом вручную. Оно часто используется при работе с пользовательским вводом, вычислениями и обработкой данных.

### Неявное преобразование

В некоторых случаях Python самостоятельно приводит типы данных к совместимому виду. Обычно это происходит во время математических операций, когда необходимо сохранить точность результата.

```python
x = 5
y = 2.5

print(x + y)
```

Результат будет иметь тип `float`, так как целое число автоматически преобразуется в число с плавающей точкой.

### Явное преобразование

Явное преобразование выполняется программистом с помощью специальных функций. Такой подход позволяет самостоятельно управлять типами данных.

```python
age = "18"

age = int(age)

print(age)
```

### Основные функции преобразования

Функция `int()` преобразует значение в целое число.

```python
print(int("25"))
```

Функция `float()` преобразует значение в число с плавающей точкой.

```python
print(float("19.99"))
```

Функция `str()` преобразует значение в строку.

```python
print(str(100))
```

Функция `bool()` преобразует значение в логический тип данных.

```python
print(bool(1))
print(bool(0))
```

### Ошибки преобразования

Некоторые операции невозможно выполнить без приведения типов. Например, нельзя напрямую сложить строку и число.

```python
result = "12" + 23
```

Такой код вызовет ошибку `TypeError`.

### Потеря данных

При некоторых преобразованиях часть информации может быть потеряна. Например, функция `int()` отбрасывает дробную часть числа.

```python
print(int(9.99))
```

Результат:

```python
9
```

### Работа с input()

Функция `input()` всегда возвращает строку, поэтому для выполнения вычислений данные обычно необходимо преобразовывать.

```python
age = int(input("Возраст: "))
```

### Практическое применение

Преобразование типов используется при обработке пользовательского ввода, выполнении математических вычислений, чтении данных из файлов и работе с различными форматами данных.

```python
name = input("Введите имя: ")
age = int(input("Введите возраст: "))

print(name)
print(age + 1)
```

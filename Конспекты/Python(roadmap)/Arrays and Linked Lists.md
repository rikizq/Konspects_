### Arrays (массивы)

Массив - это структура данных, которая хранит элементы в памяти подряд. Благодаря этому можно очень быстро получить нужный элемент по индексу. Например, если написать `arr[2]`, программа сразу найдёт третий элемент массива.

В Python чаще всего вместо классических массивов используются списки (`list`), но принцип работы у них похожий. Массивы удобны, когда нужно хранить много однотипных данных и быстро к ним обращаться.

### Что я изучил:
- как создавать массивы;
- как получать элементы по индексу;
- как изменять значения;
- как добавлять и удалять элементы;
- как проходить массив циклом;
- почему доступ по индексу работает быстро;
- какие плюсы и минусы есть у массивов.

Главный плюс массива - быстрый доступ к элементам.  
Главный минус - если вставлять элемент в середину, остальные элементы придётся сдвигать.

Пример:

```Python
numbers = [1, 2, 3, 4]print(numbers[0])numbers.append(5)numbers[2] = 10
```

### Linked Lists (связанные списки)

Связанный список - это структура данных, состоящая из отдельных элементов, которые связаны между собой ссылками. Каждый элемент хранит:
- данные
- ссылку на следующий элемент.

В отличие от массива, элементы linked list не лежат подряд в памяти. Из-за этого нельзя быстро обратиться к элементу по индексу - приходится проходить список по очереди.

### Что я изучил:
- что такое node (узел);
- как устроен связанный список;
- что такое `head`;
- как добавлять новые элементы;
- как удалять элементы;
- как проходить linked list;
- чем linked list отличается от массива;
### Какие бывают виды linked lists:
1. односвязный;
2. двусвязный;
3. циклический.

Главный плюс linked list - удобное добавление и удаление элементов.  
Главный минус - медленный поиск элементов.

Пример linked list:

```Python
a = Node(1)b = Node(2)c = Node(3)a.next = bb.next = c
```

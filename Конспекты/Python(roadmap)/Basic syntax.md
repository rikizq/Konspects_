# Python: синтаксис

Синтаксис Python определяет правила записи программ и структуру кода. Благодаря простоте и понятности язык Python широко используется в разработке программного обеспечения, анализе данных и автоматизации различных задач.

## Идентификаторы Python

Идентификатор - это имя, которое присваивается переменным, функциям, классам и другим объектам программы. Идентификатор может содержать буквы латинского алфавита, цифры и символ подчеркивания `_`, но не может начинаться с цифры. Python чувствителен к регистру, поэтому имена `name` и `Name` считаются разными идентификаторами.

Пример:

```python
user_name = "Alex"
count = 10
```

## Ключевые слова Python

Ключевые слова являются зарезервированными элементами языка и имеют специальное назначение. Они используются для создания условий, циклов, функций и других конструкций. Ключевые слова нельзя использовать в качестве имён переменных или функций.

К наиболее распространённым ключевым словам относятся `if`, `else`, `for`, `while`, `def`, `class`, `return` и `import`.

Пример:

```python
def hello():
    return "Hello"
```

## Отступы и блоки кода

Python использует отступы для выделения блоков кода. Все инструкции внутри одного блока должны иметь одинаковый уровень вложенности. После объявления условного оператора, цикла или функции ставится двоеточие `:`, после которого начинается блок кода с отступом. Обычно для оформления используется четыре пробела.

Пример:

```python
if True:
    print("Python")
```

## Типы данных и операторы

Тип данных определяет вид информации, которая хранится в переменной. Основными типами данных являются:

- `int` - целые числа;

- `float` - вещественные числа;

- `str` - строки;

- `bool` - логические значения;

- `list` - списки;

- `dict` - словари.


Операторы используются для выполнения различных действий над данными. Арифметические операторы выполняют вычисления, операторы сравнения сопоставляют значения, а логические операторы применяются при работе с условиями.

Пример:

```python
x = 10
y = 5

result = x + y
check = x > y
```

## Условные операторы, циклы и функции

Условные операторы позволяют программе принимать решения в зависимости от выполнения заданных условий. В Python для этого используются конструкции `if`, `elif` и `else`.

- `if` (если) - проверяет условие. Если условие истинно (`True`), выполняется связанный с ним блок кода.
- `elif` (иначе если) - позволяет проверить дополнительное условие, если предыдущее оказалось ложным. Таких проверок может быть несколько.
- `else` (иначе) - выполняется в том случае, если ни одно из предыдущих условий не было выполнено.

Пример:

```Python
age = 16

if age >= 18:
    print("Доступ разрешён")
elif age >= 14:
    print("Требуется сопровождение")
else:
    print("Доступ запрещён")
```

Циклы применяются для многократного выполнения команд.

- `while` (пока) - выполняет код до тех пор, пока указанное условие остаётся истинным.
- `for` (для) - используется для перебора элементов последовательности или выполнения действий заданное количество раз.

Пример:

```Python
for i in range(3):
    print(i)
```

Функция представляет собой именованный блок кода, предназначенный для решения определённой задачи.

- `def` - используется для объявления функции.
- `return` - возвращает результат работы функции в программу.

Пример:

```Python
def square(number):
    return number ** 2
```

### Введение

Циклы используются для многократного выполнения одного и того же блока кода. Они позволяют автоматизировать повторяющиеся действия, обрабатывать коллекции данных и выполнять операции до тех пор, пока выполняется определённое условие. В Python основными видами циклов являются `for` и `while`.

### Цикл for

Цикл `for` используется для перебора элементов последовательности: списков, строк, кортежей и других объектов. Также он часто применяется вместе с функцией `range()` для выполнения кода определённое количество раз. Цикл автоматически получает следующий элемент последовательности и выполняет указанный блок кода.

### Цикл while

Цикл `while` выполняется до тех пор, пока указанное условие остаётся истинным. Перед каждой новой итерацией условие проверяется повторно. Если условие становится ложным, выполнение цикла завершается. Цикл `while` удобно использовать в случаях, когда заранее неизвестно количество повторений.

### Функция range()

Функция `range()` создаёт последовательность чисел и чаще всего используется вместе с циклом `for`. Она может принимать начальное значение, конечное значение и шаг изменения. Конечное значение в диапазон не включается.

### Оператор break

Оператор `break` используется для досрочного завершения цикла. После его выполнения цикл немедленно прекращает работу независимо от оставшихся итераций. Обычно применяется при нахождении нужного значения или выполнении определённого условия.

### Оператор continue

Оператор `continue` пропускает текущую итерацию цикла и сразу переходит к следующей. Код, расположенный после `continue` внутри текущей итерации, не выполняется.

### Вложенные циклы

Вложенным называется цикл, расположенный внутри другого цикла. Внутренний цикл полностью выполняется для каждой итерации внешнего цикла. Такие конструкции используются для работы с таблицами, матрицами и многомерными данными.

### Перебор строк

Строки являются последовательностями символов, поэтому их можно перебирать с помощью цикла `for`. Во время каждой итерации в переменную цикла помещается очередной символ строки.

### Практическое применение

Циклы используются для обработки списков, строк и других коллекций данных, выполнения вычислений, поиска элементов, автоматизации повторяющихся действий и решения задач, где требуется многократное выполнение одного и того же кода.

```python
numbers = [1, 2, 3]

for number in numbers:
    print(number)

count = 0

while count < 3:
    print(count)
    count += 1
```

## Введение

Словари (`dict`) - это встроенная структура данных Python, предназначенная для хранения информации в формате «ключ - значение». В отличие от списков, доступ к данным осуществляется не по индексу, а по уникальному ключу. Благодаря этому словари позволяют быстро находить нужную информацию и эффективно работать с большими объёмами данных.

Ключи в словаре должны быть уникальными и неизменяемыми. В качестве ключей чаще всего используются строки, числа и кортежи. Значениями могут быть любые объекты Python, включая другие словари и списки.

## Тип работы

Создать словарь можно с помощью фигурных скобок `{}` или функции `dict()`. Для получения значения достаточно обратиться к соответствующему ключу. Новые элементы добавляются присваиванием, а существующие значения можно изменять. Удаление элементов выполняется с помощью оператора `del` или метода `pop()`.

Пример:

```python
user = {"name": "Анна", "role": "admin"}

print(user["name"])
```

Добавление и изменение элементов:

```python
settings = {"theme": "light", "volume": 50}

settings["volume"] = 80
settings["lang"] = "ru"
```

Удаление элементов:

```python
old_theme = settings.pop("theme")

del settings["volume"]
```

## Полезные методы

Для работы со словарями используются специальные методы:

- `keys()` - возвращает все ключи словаря;

- `values()` - возвращает все значения;

- `items()` - возвращает пары ключ - значение.


Пример:

```python
scores = {"Математика": 5, "Физика": 4}

for subject, mark in scores.items():
    print(subject, mark)
```

Проверить наличие ключа можно с помощью оператора `in`.

Пример:

```python
user = {"name": "Alex"}

if "name" in user:
    print("Ключ найден")
```

## Dictionary Comprehension

Dictionary comprehension позволяет создавать словари в компактной форме на основе циклов и условий. Такой подход делает код более кратким и удобным для чтения.

Пример:

```python
squares = {x: x ** 2 for x in range(5)}
```

## Вложенные словари

Словари могут содержать другие словари в качестве значений. Это позволяет хранить сложные структуры данных и организовывать информацию в иерархическом виде.

Пример:

```python
student = {
    "name": "Alex",
    "grades": {
        "math": 5,
        "physics": 4
    }
}
```

## Преимущества

Главным преимуществом словарей является высокая скорость поиска данных по ключу. Благодаря использованию хеш таблиц доступ к элементам выполняется практически мгновенно. Словари позволяют хранить структурированные данные, упрощают обработку информации и делают код более понятным и удобным для сопровождения.

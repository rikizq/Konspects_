### Введение

Условные конструкции позволяют выполнять разные действия в зависимости от результата проверки условия. Основными операторами являются `if`, `elif` и `else`.

### Оператор if

Оператор `if` проверяет условие. Если результат равен `True`, выполняется блок кода с отступом. Если результат равен `False`, блок пропускается.

### Группировка кода и отступы

В Python блоки кода определяются отступами. Все строки с одинаковым уровнем отступа относятся к одному блоку. Конец блока определяется уменьшением отступа. Благодаря этому Python не использует фигурные скобки.

### Оператор else

Оператор `else` выполняется, если условие `if` оказалось ложным. Он не содержит собственного условия и используется для альтернативного варианта выполнения.

### Оператор elif

Оператор `elif` используется для проверки дополнительных условий. Проверка выполняется сверху вниз. После нахождения первого истинного условия остальные проверки пропускаются.

### Вложенные условия

Условные конструкции могут находиться внутри других условий. Для каждого нового уровня используется дополнительный отступ.

### Логические выражения

В условиях часто используются операторы сравнения `==`, `!=`, `>`, `<`, `>=` и `<=`. Для объединения нескольких условий применяются операторы `and`, `or` и `not`.

### Практическое применение

Условные конструкции используются для проверки данных, выбора действий программы и обработки различных ситуаций во время выполнения кода.

```python
temperature = 22

if temperature > 25:
    print("Жарко")
elif temperature > 15:
    print("Тепло")
else:
    print("Прохладно")
```

### Введение

Множества (`set`) в Python - это изменяемые коллекции уникальных элементов. Они автоматически удаляют дубликаты и не поддерживают индексацию. Благодаря особенностям внутреннего хранения множества обеспечивают очень быстрый поиск элементов и часто используются при работе с большими объёмами данных.

### Создание множеств

Множество можно создать с помощью фигурных скобок `{}` или функции `set()`. Для создания пустого множества необходимо использовать только `set()`, поскольку пустые фигурные скобки создают словарь.

### Основные методы

Для добавления элемента используется метод `add()`. Удаление выполняется с помощью `remove()` или `discard()`. Метод `update()` добавляет элементы из другой коллекции. Метод `copy()` создаёт копию множества, а `clear()` полностью очищает его содержимое.

### Операции над множествами

Операция объединения (`union()` или `|`) возвращает все уникальные элементы. Пересечение (`intersection()` или `&`) возвращает только общие элементы. Разность (`difference()` или `-`) возвращает элементы первого множества, отсутствующие во втором. Симметричная разность (`symmetric_difference()`) возвращает элементы, которые присутствуют только в одном из множеств.

### Подмножества и проверка принадлежности

Операторы `in` и `not in` используются для проверки наличия элемента. Методы `issubset()` и `issuperset()` позволяют проверить отношения подмножества и надмножества между двумя множествами.

### Практическое применение

Множества используются для удаления дубликатов, поиска уникальных значений, сравнения наборов данных и быстрого поиска элементов. Они особенно полезны при обработке больших коллекций информации, где важна скорость выполнения операций.

```python
numbers = {1, 2, 3}

numbers.add(4)
numbers.remove(2)

print(3 in numbers)
```

```python
a = {1, 2, 3}
b = {2, 3, 4}

print(a | b)
print(a & b)
print(a - b)
```
